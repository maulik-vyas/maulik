Data Science Projects

Salary Prediction Project

Developed a machine learning-based salary prediction tool by scraping over 1 million job postings.
Engineered features from job descriptions and implemented models such as Linear, Polynomial, Gradient Boosting, and Ridge Regression.
Optimized models using 5-fold cross-validation and automated the pipeline for real-time salary predictions, providing data-driven insights for employee compensation and workforce planning.

Credit Card Fraud Detection

Developed a fraud detection model for bank transactions using a highly imbalanced dataset with 492 frauds out of 284,807 transactions.
Applied PCA to transform features and utilized Logistic Regression for binary classification, achieving a 0.995 ROC-AUC score.
The model accurately predicts fraudulent transactions, helping banks prevent losses by identifying suspicious activity in real time.

Employee Turnover Prediction

Developed a machine learning model to predict employee turnover using four datasets: original, upsampled, SMOTE, and downsampled.
Applied SMOTE to address class imbalance and used Logistic Regression, Random Forest, and Gradient Boosting models.
After extensive EDA and model evaluation, Random Forest Classifier achieved the highest performance with a 0.99 F1-score, enabling more effective employee retention strategies and decision-making.

Bank Customer Churn Prediction

Developed a machine learning model to predict customer churn for a multinational bank using a dataset from Kaggle.
Applied Random Forest Classifier, Decision Tree Classifier, and Logistic Regression, with Random Forest achieving the best accuracy.
Built a Streamlit app to provide a user-friendly interface for real-time churn predictions based on customer data, supporting proactive retention strategies.

Bank Loan Approval Prediction

Developed a machine learning model to predict loan approval for customers using a Kaggle dataset.
Applied Random Forest, Decision Tree, and Naive Bayes algorithms, with Naive Bayes yielding the best performance.
Preprocessed the data by converting categorical variables and removed unnecessary columns.
Trained the model, achieving 0.87 accuracy, and deployed it as a web app using Streamlit for real-time loan approval predictions.

Health Insurance Cost Prediction

Developed a model to predict health insurance costs using various features of individuals.
Addressed the regression problem by applying Linear Regression, Support Vector Regressor, Random Forest Regressor, and Gradient Boosting Regressor.
After preprocessing the data with One-Hot Encoding and MinMaxScaler, Gradient Boosting Regressor performed best, achieving the lowest Mean Absolute Error (MAE = 2407.27), providing accurate insurance cost predictions.

Image Classification by Transfer Learning

Developed a food classification model using the Food 101 dataset, focusing on 10 food classes.
Applied data augmentation to improve model robustness and used transfer learning with EfficientNetV2B0 and ResNet50.
After creating a feature extraction model and fine-tuning the top layers of EfficientNetV2B0, the model achieved 91.68% accuracy, successfully classifying food images with enhanced performance.

Skim Literature using Pretrained Sentence Encoder

Developed an NLP model to classify research paper abstracts into sections (Background, Objective, Method, Result, Conclusion) for efficient skimming.
Using a dataset of 20,000 abstracts, I employed a pretrained Universal Sentence Encoder and experimented with various models, including Conv1D and hybrid embedding layers.
The best-performing model, a tribrid embedding, achieved 82.57% accuracy and 0.82 F1-score on the test set, enabling effective abstract segmentation.
